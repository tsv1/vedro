## Installation

```shell
$ pip3 install vedro
```

## Usage

```python
# ./scenarios/decode_base64_encoded_string.py
import base64
import vedro

class Scenario(vedro.Scenario):
    subject = "decode base64 encoded string"

    def given(self):
        self.encoded = "YmFuYW5h"

    def when(self):
        self.decoded = base64.b64decode(self.encoded)

    def then(self):
        assert self.decoded == b"banana"
```

```shell
$ vedro run
```
